### ✔️ 목표

* 스프링 프레임워크 핵심 기술 강의를 통해 스프링 프레임워크를 이해한다.
* 주말 제외, 계획한 분량을 학습할 수 있도록 노력한다.
* 학습한 내용은 [TIL](https://github.com/weekyeon/til)에 기록한다.
* 정해놓은 기간에 치우쳐 **대충** 학습하지 않도록 한다.
* 이해가 되지 않는 부분은 **반복**해서 듣거나 좀 더 깊게 공부하도록 한다.




### ✔️ 목차

* SpEL
